Logo Clouds

These logo clouds are designed to be responsive, and have options for logo sizing.

Sizing

Logo sizing is managed with the .logo-size--[small|medium|large], attached to each logo image. This allows flexible sizing options, without having to style each individually when there are aspect ratio variations. The size classes manage the height of the image, and have responsive adjustments for different viewports.

Leverage Middleman Data Structures

Use the example Middleman loop in this page, with YAML data for easy management of custom logo clouds. They typically have a name, web url, filename, and can have per-item custom classes (great for sizing variations on odd aspect ratios).

Flexbox Classes

The .logo-cloud class contains the objects with .logo-cloud--item, and uses flexbox for positioning. Use .logo-size--[small|medium|large] for sizing, and if you want grayscale logos with a CSS filter, apply .logo-style--monochrome to the tag.

Accessibility is maintained through using alt title attributes, if the images are links (Read more on accessibility).
